The problem:

I have a query that should return 1-3 values. I can access the first one, but no chance with the second and so on. I want to be able to print the result in a simple drop-down list.

exising code sample:
$sql = mysql_query("SELECT category FROM categories ORDER BY category");
$result = mysql_fetch_assoc($sql);

echo "\n";

The solution:
Rewrite the code as this and I think you will understand better what is happening.
$sql = "SELECT category FROM categories ORDER BY category";
$result = mysql_query($sql);
while ( $row = mysql_fetch_assoc($result)) {
       echo "\n";
}

NOTE: mysql_fetch_assoc either returns an associative array corresponding to the fetched row or it will
return FALSE if there are no more rows to fetch.
while ($result = mysql_fetch_assoc($sql)) will do whatever you tell it to do with each row that is returned,
once the end is reached (i.e. mysql_fetch_assoc($sql) returns FALSE) the next part of your script will be executed.
