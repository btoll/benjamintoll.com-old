tar Cheat Sheet


tar + bzip2
tar cjvf my.tbz2 /dir

tar + gzip
tar czvf my.tgz /dir

decompress + unpack bzip2
tar xjvf my.tbz2

decompress + unpack gzip
tar xzvf my.tgz

append files to the end of an archive (note this is a tarball, it hasn't been compressed)
tar rf my.tar anotherfile.js

list the contents of an archive (note this is a tarball, it hasn't been compressed)
tar tf my.tar

extract a file from an archive (note this is a tarball, it hasn't been compressed)
tar xvf {my.tar} {file_to_extract}

extract a directory from an archive (note this is a tarball, it hasn't been compressed)
tar xvf {my.tar} {dir_to_extract}
