I found that I needed to set the date one day in the future, so this is what I did:


var oNextDay = new Date();
oNextDay.setTime(oNextDay.getTime() + 86400000);
		
//will output Sat Jul 19 2008 14:03:51 GMT-0400 (Eastern Daylight Time)

You can see that since the Unix timestamp is being used (calculated in milliseconds), the months will automatically "roll-over":


var oNextDay = new Date(Date.parse("July 31, 2008"));
oNextDay.setTime(oNextDay.getTime() + 86400000);

//will output Fri Aug 01 2008 00:00:00 GMT-0400 (Eastern Daylight Time)

*NOTE* Different browsers will output the Date slightly differently when the Date object's toString() method is invoked.
