Need to send along authentication credentials when making a REST submission via Ajax?

Want to avoid receiving a HTTP Error 401 and then the dreaded Basic HTTP Authentication dialog box?

It's as easy as setting a header and encoding the username:password in Base64.


oXHR.open("GET", sURL, false);
oXHR.setRequestHeader("Authorization", "Basic YWRtaW4="); //"YWRtaW4=" is the Base64 encoded username:password;

Note the XMLHttpRequest object's open method must be invoked before any headers are set.
