Single Click, Different Alignment
  • This demo uses the forest theme.
  • Click the '...' button to display the calendar.
  • The calendar pops up above the button instead of below it.
  • In the calendar, select a date with a single click.
  • The results are inserted in the following format:
    day/month/four digit year (i.e.: 01/01/2007)
