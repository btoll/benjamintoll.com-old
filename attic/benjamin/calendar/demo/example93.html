Select Multiple Days in a Flat Calendar
  • This demo uses the yellow theme.
  • This is a flat in-the-page calendar. It is displayed when the page loads rather than when the user clicks on a button.
  • You can now select multiple days in the calendar.
  • You can click a selected date to remove it from the list of selected dates.
  • The following dates have been programmatically selected:
    • 2004 April 25
    • 2004 April 18
    • 2004 April 22
    • 2004 April 05
    You can uncheck these dates.
  • Click the Submit button to display the selected dates in the text area.
  • Notice how the title dynamically changes (titleHtml option).
